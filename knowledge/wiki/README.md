# knowledge/wiki

This directory is for reusable project knowledge distilled from original materials and experiments.

Use this directory for:

- Standard-to-feature mapping notes
- Scene feature taxonomy
- Definitions and formulas used repeatedly in the thesis
- Summaries of papers, standards, and expert feedback
- Stable conclusions from experiment review

Do not place original PDFs, DOCX, PPTX, CAJ files, or other raw source materials here. Put those in `knowledge/sources/`.

Current status: initialized only. No reusable wiki article has been added yet.

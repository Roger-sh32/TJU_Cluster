# logs

This directory is for process records generated while working on the project.

Use this directory for:

- Experiment run logs
- Error records and fixes
- AI collaboration notes
- User or expert feedback
- Data quality findings
- Reproducibility notes, including commands and environment details

Do not place final thesis deliverables here. Put final reports, slides, and shareable outputs in `outputs/`.

Current status: initialized only. No project-wide execution log has been added yet.
